# Fearless Fluent GUI

## Tasks

- Study the existent standard library of Fearless both in the current
  incarnation and the old legacy version.
- Design a Fluent GUI building on top of SWING. This GUI will work both on
  Java and on Fearless.
- Design a range of test applications boht in Java and Fearless using this
  GUI library, crucially covering animation and real time interactions.

## Part 1

- Design and Implement the Java version for the GUI library.
- Produce some simple application using this library in Java.
- Produce a report explaining the design. This report should double as a
  guide and documentation.

*Animation Aspect not Required*

## Part 2
    
- Integrate animation support and key bindings.
- Integrate the GUI library in Fearless. 
- Demonstrate this GUI library in a few interesting Fearless programs.
- Produce a report explaining the complete design. This report should
  double as a guide and documentation.
- The design ought to respect the Fearless safety properties by correctly
  levering Object capabilities, and the report should discuss what is
  observable at the various layers of the applications.

## Part 3

- Final presentation.
- Showcase applications and examples showing the applicability of the Fluent GUI.
